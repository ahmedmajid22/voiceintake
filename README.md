# VoiceIntake
AI-powered voice intake for healthcare clinics.
Patient speaks naturally → Whisper transcribes → AI extracts → Form fills automatically.

## Quick Start
./start.sh
Open http://localhost:5173

## Stack
- Speech-to-Text: OpenAI Whisper
- AI Extraction: Groq / Llama 3.3
- Form Filling: Playwright
- Backend: Python + FastAPI
- Frontend: React + Tailwind
